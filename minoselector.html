Initial settings customizer

  

Add custom garbage to the start of your 2-player game.

Select the mino types from the right grid and add them to the green cells of the left grid - This will be your initial playfield.

To select a minotype:

  • Click it
  • Use keys A-Z or 0-9 or . and - to select the corresponding minotypes
  • Use numpad 1-8 to select the standard garbage minotypes
You can also make the minotype random by checking the checkbox below.

Load a board
You can load a playfield by copying a CSV file to the Textarea below
or by choosing from pre-defined boards (dropdown selection).

Info:
The stack is only fully rendered if you chose height 5 in-game.
Only the top parts are rendered if you chose heights 1-4 and nothing is shown with height 0.
If any row of your stack is full, it will clear immediately once the game starts.
