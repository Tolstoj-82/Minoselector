Customize the initial garbage

Add custom garbage to the start of your 2-player game.

Select the mino types from the right grid and add them to the green cells of the left grid. This will be your initial playfield.
To select a minotype either click it
or use keys A-Z and 0-9 to select the corresponding mino types.
or use numpad 1-8 to select the standard mino types for the garbage
You can also make the minotype random by checking the checkbox below.

Info:
The full stack is rendered if you chose height 5 whereas only the top parts are rendered if you chose height 1-4.
If any row of your garbage is full, it will clear the line immediately.
